الخطوة 1️⃣: تشغيل الإعداد التلقائي
──────────────────────────────────────────────────────────────
  1. افتح الملف: AUTO_SETUP.html في المتصفح
  2. سيبدأ الإعداد تلقائياً (30-60 ثانية)
  3. انسخ Database ID و Collection ID
  4. مثال:

الخطوة 2️⃣: تحديث ملف الإعدادات
──────────────────────────────────────────────────────────────
  1. افتح ملف: config-preset.js في محرر نصوص
  2. ابحث عن السطور (9 و 10):
  
     databaseId: '',
     collectionId: ''
  
  3. غيرها إلى:
  
     databaseId: 'ضع Database ID هنا',
     collectionId: 'ضع Collection ID هنا'
  
  4. احفظ الملف


الخطوة 3️⃣: رفع على GitHub
──────────────────────────────────────────────────────────────

  ═══ الطريقة السهلة (GitHub Desktop) ═══
  
  1. حمل GitHub Desktop من: desktop.github.com
  2. ثبت البرنامج وسجل دخول
  3. File → New Repository
     - Name: phc-leaflet-generator
     - Local Path: اختر المجلد الحالي
     - Create Repository
  
  4. اضغط "Publish repository"
     - اختر Public
     - Publish
  
  5. افتح المشروع على GitHub.com
     - Settings → Pages
     - Source: main branch
     - Folder: / (root)
     - Save
  
  6. انتظر 2-3 دقائق
  
  7. الموقع سيكون على:
     https://username.github.io/phc-leaflet-generator/
  
  
  ═══ الطريقة البديلة (Git Command Line) ═══
  
  افتح Terminal/Command Prompt في مجلد المشروع:
  
  git init
  git add .
  git commit -m "Initial commit: PHC Leaflet Generator"
  git remote add origin https://github.com/USERNAME/phc-leaflet-generator.git
  git branch -M main
  git push -u origin main
  
  ثم فعل GitHub Pages من Settings → Pages
