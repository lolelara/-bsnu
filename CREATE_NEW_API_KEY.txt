🔧 الحل السريع (5 دقائق):
═══════════════════════════════════════════════════════════════════

الخطوة 1: اذهب لإنشاء API Key
──────────────────────────────────────────────────────────────

1. افتح: https://makersuite.google.com/app/apikey
2. سجل دخول بحساب Google
3. اضغط "Create API Key"
4. اختر "Create API key in new project"
5. انسخ الـ API key الجديد

الخطوة 2: حدث الملف
──────────────────────────────────────────────────────────────

1. افتح: config-preset.js
2. غير السطر 5:
   
   geminiApiKey: 'YOUR_NEW_API_KEY_HERE'

3. احفظ الملف (Ctrl+S)

الخطوة 3: جرب الموقع
──────────────────────────────────────────────────────────────

1. افتح Private Window: Ctrl + Shift + N
2. اسحب index.html للنافذة الجديدة
3. أدخل رقم طالب: 1
4. اضغط Generate
5. يشتغل! ✓
