## 📋 نظرة عامة

مولد المطويات الصحية هو تطبيق ويب مصمم خصيصاً لطلاب السنة الثالثة في **كلية الطب - جامعة بني سويف الأهلية**. يساعد التطبيق الطلاب على إنشاء مطويات صحية احترافية لنشاط الرعاية الصحية الأولية (PHC) بسهولة وسرعة.

## ✨ المميزات الرئيسية

### 🎨 التصميم والواجهة
- ✅ **تصميم عصري** مع Glass Morphism وتأثيرات متحركة
- ✅ **20 ثيم مختلف** للاختيار من بينها
- ✅ **واجهة عربية** كاملة مع دعم RTL
- ✅ **تصميم متجاوب** يعمل على جميع الأجهزة
- ✅ **رسوم متحركة سلسة** لتحسين تجربة المستخدم

### 📄 إنشاء المطويات
- ✅ **توليد محتوى تلقائي** باستخدام Gemini AI
- ✅ **صور عالية الجودة** يتم إنشاؤها بالذكاء الاصطناعي
- ✅ **19 موضوع صحي** مختلف
- ✅ **تخصيص تلقائي** حسب رقم الطالب التسلسلي

### 📥 التصدير والطباعة
- ✅ **تحميل PDF محسّن** مع دعم كامل للعربية
- ✅ **طباعة مباشرة** بجودة عالية
- ✅ **حفظ الصور** بصيغ متعددة
- ✅ **تسمية تلقائية** للملفات

### 🤖 أدوات الذكاء الاصطناعي
- ✅ **تبسيط المحتوى** للجمهور العام
- ✅ **توليد النقاط الرئيسية** تلقائياً
- ✅ **الاستماع للمحتوى** بتقنية TTS
- ✅ **تحديث المحتوى والصور** بضغطة زر

## 🚀 البدء السريع

### المتطلبات
- متصفح ويب حديث (Chrome, Firefox, Edge, Safari)
- اتصال بالإنترنت
- مفتاح API من Google Gemini (للمطورين)

### الاستخدام

1. **افتح التطبيق**
   ```
   https://lolelara.github.io/-bsnu
   ```

2. **أدخل رقمك التسلسلي** (من 1 إلى 644)

3. **انقر على "إنشاء المطوية"**

4. **انتظر** حتى يتم توليد المحتوى والصور

5. **اختر الثيم المفضل** من بين 20 تصميم

6. **حمّل المطوية** كملف PDF أو اطبعها مباشرة

## 🛠️ التقنيات المستخدمة

### Frontend
- **HTML5** - البنية الأساسية
- **CSS3** - التصميم والأنيميشن
- **JavaScript (ES6+)** - المنطق والتفاعل
- **Tailwind CSS** - إطار عمل CSS

### APIs & Services
- **Google Gemini AI** - توليد المحتوى
- **Imagen 3.0** - توليد الصور
- **Gemini TTS** - تحويل النص لصوت

### Libraries
- **html2pdf.js** - تحويل HTML إلى PDF
- **html2canvas** - التقاط لقطات الشاشة
- **jsPDF** - إنشاء ملفات PDF
- **Font Awesome** - الأيقونات

## 🔧 التثبيت للمطورين

### 1. استنساخ المشروع
```bash
git clone https://github.com/lolelara/-bsnu.git
cd -bsnu
```

### 2. إضافة مفتاح API
افتح `New Text Document.html` وأضف مفتاح Gemini API الخاص بك:
```javascript
const apiKey = "YOUR_GEMINI_API_KEY_HERE";
```

### 3. تشغيل المشروع
افتح `New Text Document.html` في المتصفح أو استخدم خادم محلي:
```bash
# باستخدام Python
python -m http.server 8000

# باستخدام Node.js
npx http-server
```

## 🐛 حل المشاكل الشائعة
