═══════════════════════════════════════════════════════════════════
   🔄 مشكلة Cache - الحل السريع
═══════════════════════════════════════════════════════════════════

المشكلة:
  المتصفح يستخدم النسخة القديمة من الملفات (cached)
  
الحل:
  Hard Refresh لتحميل الملفات الجديدة

═══════════════════════════════════════════════════════════════════
🎯 الحل السريع (جربها بالترتيب):
═══════════════════════════════════════════════════════════════════

الطريقة 1️⃣: Hard Refresh
──────────────────────────────────────────────────────────────

Windows:
  Ctrl + Shift + R
  أو
  Ctrl + F5

Mac:
  Cmd + Shift + R
  أو
  Cmd + Option + R


الطريقة 2️⃣: Clear Cache من DevTools
──────────────────────────────────────────────────────────────

1. اضغط F12 (لفتح DevTools)
2. اضغط بالزر الأيمن على زر Refresh 🔄
3. اختر "Empty Cache and Hard Reload"


الطريقة 3️⃣: Clear Cache يدوياً
──────────────────────────────────────────────────────────────

Chrome/Edge:
  1. Ctrl + Shift + Delete
  2. Time range: Last hour
  3. ✓ Cached images and files
  4. Clear data
  5. F5

Firefox:
  1. Ctrl + Shift + Delete
  2. Time range: Last hour
  3. ✓ Cache
  4. Clear
  5. F5


الطريقة 4️⃣: Incognito/Private Mode
──────────────────────────────────────────────────────────────

1. اضغط Ctrl + Shift + N (Chrome) أو Ctrl + Shift + P (Firefox)
2. افتح index.html في النافذة الجديدة
3. جرب التوليد


الطريقة 5️⃣: Disable Cache في DevTools
──────────────────────────────────────────────────────────────

1. اضغط F12
2. اذهب إلى Network tab
3. ✓ Disable cache
4. أبقي DevTools مفتوح
5. F5
6. جرب التوليد

═══════════════════════════════════════════════════════════════════
✅ بعد أي طريقة من فوق:
═══════════════════════════════════════════════════════════════════

1. افتح Console (F12 → Console)
2. أدخل رقم: 1
3. اضغط Generate
4. راقب Console - لازم تشوف:
   ✓ No more "gemini-1.5-flash" errors
   ✓ Using "gemini-1.5-flash-latest" ✓

═══════════════════════════════════════════════════════════════════
🔍 للتأكد أن الملف محدث:
═══════════════════════════════════════════════════════════════════

1. افتح Console (F12)
2. اكتب:
   
   fetch('main-app.js').then(r => r.text()).then(t => console.log(t.includes('gemini-1.5-flash-latest')))

3. لو طلع: true → الملف محدث ✓
4. لو طلع: false → جرب hard refresh تاني

═══════════════════════════════════════════════════════════════════
💡 نصيحة:
═══════════════════════════════════════════════════════════════════

أسهل طريقة:
  Ctrl + Shift + R (Windows)
  Cmd + Shift + R (Mac)

لو ما اشتغلت:
  Incognito Mode (Ctrl + Shift + N)
